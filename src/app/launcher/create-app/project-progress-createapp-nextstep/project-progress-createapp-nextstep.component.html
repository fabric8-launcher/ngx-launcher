Next Steps: Update your booster using Continuous Delivery

Your booster is automatically configured to build and deploy with new commits.

  1. Clone your project from GitHub.

    $ git clone {{getProgressByKey('GITHUB_CREATE').hyperText}}

  2. Open your project in your desired IDE or editor.

  3. Perform any updates you want to the project.

  4. Commit and push your changes back to GitHub.

    $ git add .
$ git commit -m "Made an update"
$ git push

When changes are pushed to your GitHub repository, a new build is automatically triggered on your OpenShift project with those changes.
